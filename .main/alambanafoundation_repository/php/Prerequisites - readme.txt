
Prerequisites:
**Ensure you have XAMPP installed on your computer. You can download XAMPP from here.

Steps:
1. Download the Website Files:
Download the website files from the source.
Extract the files to a folder on your computer.

2. Move Website Files to XAMPP's htdocs Directory:
Move the extracted website folder to C:\xampp\htdocs\.
For example, if your website folder is named alambana, the path should be C:\xampp\htdocs\alambana.

3. Start XAMPP:
Start XAMPP by running xampp-control.exe from your XAMPP installation directory (usually C:\xampp).
Start the Apache server by clicking the "Start" button next to "Apache" in the XAMPP Control Panel.

4. Access the Website:
Open your web browser.
Visit http://localhost/alambana/.

Now, you should be able to access the alambana website locally on your system using XAMPP. Please make sure that the website files are structured correctly within the alambanafoundation.org folder inside the htdocs directory.
